# Quick Start Guide

## Installation & Running

```bash
# 1. Install dependencies
npm install

# 2. Run development server
npm run dev

# 3. Open http://localhost:3000/dashboard
```

## What You'll See

- **Dashboard Page** (`/dashboard`): Full analytics dashboard with KPIs and charts
- **Filters**: Change date range and user type to see data update
- **Theme Toggle**: Click moon/sun icon in header
- **Export CSV**: Click "Export CSV" button
- **Mobile Menu**: Resize browser to see responsive design

## Key Features to Test

1. **Filter Changes**: Select different date ranges and user types
